📘 About This Project

🚀 Overview

This system performs vehicle classification using Deep Learning and supports both image upload and real-time camera detection. It integrates a confidence-based decision layer for better prediction interpretation.

⚡ Features
  • Upload image classification
  • Live camera detection (real-time)
  • Audio feedback (speech + ambulance siren)
  • Confidence-based decision system
  • Interactive UI with animations
📁 Dataset

Data collected from Kaggle and Zenodo to ensure diversity.

  • Total Images: ~2000+
  • Classes: Car, Bike, Truck, Bus, Ambulance
  • Ambulance treated as priority class
🧹 Preprocessing
  • Resize to 224×224
  • Normalization (0–1 scaling)
  • Augmentation (flip, rotation, zoom)
  • Dataset cleaning
