📘 About This Project

🚀 Project Overview

This system classifies vehicle images using Deep Learning. It can detect Car, Bike, Truck, Bus, and Ambulance with high accuracy and includes a confidence-based decision system.

📁 Dataset Source

The dataset was collected from multiple sources including Kaggle and Zenodo to ensure diversity and better generalization.

📊 Dataset Details
  • Total Images: ~2000+
  • Classes: 5 (Car, Bike, Truck, Bus, Ambulance)
  • Car: ~350–400 images
  • Bike: ~350–400 images
  • Truck: ~350–400 images
  • Bus: ~300–400 images
  • Ambulance: ~500+ images
🖼 Image Resolution & Format

Images were collected in different resolutions ranging from low resolution (200×200) to high resolution (1920×1080). Formats included JPG and PNG. All images were standardized during preprocessing.

🧹 Preprocessing Steps
  • Resized all images to 224×224
  • Normalized pixel values (0–255 → 0–1)
  • Applied data augmentation (rotation, zoom, flip)
  • Split dataset into 80% training and 20% validation
  • Removed noisy and incorrect samples
